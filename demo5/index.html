css如何实现水平居中

一、使用margin:auto属性

我是居中的div，使用的是margin:auto属性.代码如下：
                
                .item1{
                    width: 1200px;
                    height: 50px;
                    background-color: #25aacd;
                    margin: auto;
                }

二、设置margin-left: （父宽度-本身宽度）/2属性（已知父元素和本身宽度）

我是居中的div，我知道我的父亲有多宽，我也知道我自己有多宽，并且我们宽度的单位是一致的。代码如下： 
                
                .item2{
                    width: 90%;
                    height: auto;
                    border: 1px solid #25aacd;
                    padding: 25px;
                    margin-left: 5%;
                    box-sizing: border-box;
                }

三、position属性

我是使用相对定位居中的div,具体代码如下： 
                    
                    .item3{
                        position: relative/absolute/inherit;
                        width: 1200px;
                        height: auto;
                        left: 50%;
                        margin-left: -600px;
                        border: 1px solid #25aacd;
                        padding: 25px;
                        box-sizing: border-box;
                    }

四、flex属性值实现

我要居中，需要设置我的父亲的css属性。代码如下： 
                    
                        .item4{
                            display: flex;
                            justify-content: center;
                        }
                        .item4 .item{
                            width: 1000px;
                            height: auto;
                            border: 1px solid #25aacd;
                            padding: 25px;
                            box-sizing: border-box;
                        }
