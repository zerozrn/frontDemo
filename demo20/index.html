灵活的背景定位

场景: 很多时候, 我们想针对某个容器某个角对背景图片做偏移定位, 如右下角,在CSS2.1中只能针对容器左上角偏移, 若想图片和容器的
边角之间留有一定的缝隙,该如何实现呢?对于尺寸固定的容器,不难, 但是容器宽高往往是变化的,这时候就需要css3了.

(1)background-position扩展语法方案

Code Pirate 
    
        background: url("../imgs/logo.png")
                    no-repeat bottom right #25aacd; /* 回退方案,浏览器兼容性 */
        background-position: right 20px bottom 10px;  /*  css3扩展语法,指定方向上的偏移量

(2)background-origin方案

场景: 假如偏移量与容器的内边距一致,又该怎么做呢?

Code Pirate 
    
        background: url("../imgs/logo.png") no-repeat #25aacd
                    bottom right;
        background-origin: content-box;

(3)calc()函数方案

Code Pirate 
    
        background: url("../imgs/logo.png") no-repeat #25aacd;
        background-position: calc(100% - 20px) calc(100% - 10px);
