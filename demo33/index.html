弹性布局实现垂直居中元素

可能用到的properties有justify-content、align-items、align-self，且跟轴的方向有关系，轴的方向不同，方法可能会不同。

1 水平居中（父元素宽高未知，项目宽度已知，单项目）

I am horizontally centered.

This box is horizontally centered. Even if the text in this box changes to make it wider or taller, the box will still be centered. Go ahead, give it a try. Just click to edit the text.

关键代码如下：

    
        /* 1、这是第一种方法 */
        .container{
            display: flex;
            justify-content: center;
        }
        .child{
            /* 为了阻止子元素撑开整个父元素，设置其位置 */
            align-self: flex-start;
        }

        /* 2、这是第二种方法 */
        .container{
            display: flex;
            flex-direction: column;
        }
        .child{
            align-self: center;
        }

4 水平垂直居中

I am horizontally and vertically centered

This box is both vertically and horizontally centered. Even if the text in this box changes to make it wider or taller, the box will still be centered. Go ahead, give it a try. Just click to edit the text. 
    
        /* 1、这是第一种方法 */
        .container{
            display: flex;
            justify-content: center;
            align-items: center;
            /* 轴的方向改变不影响结果 */
            /* flex-direction: column; */
        }

        /* 2、这是第二种方法 */
        .container{
            display: flex;
            justify-content: center;
             /* 轴的方向改变不影响结果 */
            /* flex-direction: column; */
        }
        .child{
            align-self: center;
        }
